# Notebooks

Jupyter notebooks for experiments and tutorials.
These run in Google Colab or VS Code with the Jupyter extension.

## Structure
- `experiments/` - Neural network experiments
- `tutorials/`   - Step by step learning notebooks (Year 3+)

## How to use in Colab
1. Upload notebook to Google Drive
2. Open with Google Colab
3. Run `!git pull origin master` to get latest code
